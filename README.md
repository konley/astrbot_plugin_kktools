# astrbot_plugin_kktools（AI工具箱）

AstrBot 版「AI工具箱」插件，移植[@渔火](https://github.com/yhArcadia)的 `ai-tools.js`。

省流总结、识图、阴阳怪气反话、地理定位的插件来源是渔火写的。

## 功能一览

| 功能 | 触发方式 | 说明 |
|------|----------|------|
| **省流总结** | `省流` / `总结` / `tldr` 开头 | 引用消息/合并转发/直接文本，用大模型提炼核心信息，限 300 字 |
| **识图** | `识图` 开头 | 八大类分类体系识别图片内容，支持 @头像识别 |
| **阴阳怪气** | `好吃吗` / `好用吗` 等结尾 | 对任何东西说反话，贬低但显得理中客，限 100 字 |
| **地理定位** | `在哪` 开头 | OSINT 地理位置分析，四步推理输出结构化排名 |

## 使用方法

1. **省流总结**：发送 `省流` + 引用一条消息（支持文本/图片），或直接 `省流 今天发生了什么大事`
2. **识图**：发送图片 + `识图`，或 `识图` + 引用含图消息，或 `@某人 识图` 识别头像
3. **阴阳怪气**：发送 `这个东西好吃吗` / `那个游戏好玩吗` 等
4. **地理定位**：发送图片 + `在哪`，或 `在哪` + 引用含图消息

> 所有关键词均可在 WebUI 中自定义，支持 `/` 和 `#` 前缀（可开关）。

## 配置项

在 AstrBot WebUI → 插件管理 → AI工具箱 中配置：

### 全局设置

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `ignore_slash` | bool | `true` | 忽略指令前的 `/` 或 `#` 前缀 |
| `enable_vision` | bool | `true` | 启用图片分析（需 Provider 支持多模态） |
| `cooldown` | int | `5` | 用户冷却时间（秒） |

### 各功能配置

每个功能均可独立开关、自定义触发关键词（支持多个）和系统提示词：

- **省流总结**：关键词默认 `省流 / 总结 / tldr / TLDR`（前缀匹配）
- **识图**：关键词默认 `识图`（前缀匹配）
- **阴阳怪气**：关键词默认 `好吃吗 / 好喝吗 / 好用吗 / 好玩吗 / 好看吗`（后缀匹配，允许 `?`/`？`）
- **地理定位**：关键词默认 `在哪`（前缀匹配）

> 系统提示词留空则使用内置默认（来自原版 ai-tools.js）。

## 与原版的区别

- 使用 AstrBot 内置 LLM Provider，无需手动配置 API URL/Key
- 合并转发消息由 AstrBot 适配器自动解析，无需手动处理 XML/JSON
- 关键词全部 WebUI 可配置，支持 `/` `#` 前缀开关
- 去除了博查搜索、表情回应等平台特有功能，聚焦核心 LLM 能力
